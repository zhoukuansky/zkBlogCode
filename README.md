# zkBlogCode
### 前言：  
>最近几天笔者在搭建一个个人博客网站。由于作者有自己的服务器，本着不浪费资源的态度，于是就在其上部署个博客网站。虽然使用学生优惠申请的服务器（最低配），但用来更新下自己的博客还是绰绰有余的。
>决定了要搭建自己的个人博客之后，就面临着如何开发的问题了。自己开发个博客网站，外加管理员的页面？想想就觉得工程量不适合我这种Lazy Boy。那就用现成的框架吧。经过一番比较之后，笔者选用了Tale博客框架。
